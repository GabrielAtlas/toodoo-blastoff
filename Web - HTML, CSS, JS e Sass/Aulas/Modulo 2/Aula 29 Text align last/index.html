O Motor a vapor de Newcomem foi inventado por Thomas Newcomen em 1712, muitas vezes chamado como Máquina atmosférica Newcomen. O motor é operado pelo vapor de condensação introduzido no cilindro, criando assim um vácuo parcial, permitindo assim que a pressão atmosférica empurre o pistão para dentro do cilindro.
